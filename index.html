DEV211.1x - Introduction to HTML and Javascript - Microsoft

This is a Microsoft course on edX. It includes information on the most common HTML and CSS concepts. It also covers many of the new things in HTML5. Some of what is included is Semantic Elements, SVG, Canvas and Media Controls. It covers basic and advances JS including callbacks and the HTML5 APIs. It then gives a brief introduction to external libraries such as jQuery, AngularJS, Bootstrap, Typescript and Sass.
